Web apis
===========

As Web APIs são conjuntos de ferramentas e funcionalidades integradas diretamente nos navegadores que expandem significativamente as capacidades do JavaScript.

Na prática, as Web APIs se dividem em várias categorias especializadas. Temos APIs de armazenamento como LocalStorage e IndexedDB para guardar dados no navegador; APIs de comunicação como Fetch e WebSockets para trocar informações com servidores; APIs de interface como History e Fullscreen para controlar a experiência do usuário; e APIs de hardware como Geolocation, Camera e Device Orientation para acessar recursos físicos do dispositivo.

O que torna essas APIs particularmente poderosas é sua natureza assíncrona. A maioria opera em segundo plano, permitindo que a interface do usuário permaneça responsiva enquanto tarefas como carregar dados ou processar arquivos acontecem paralelamente. Além disso, elas seguem rigorosos protocolos de segurança, exigindo permissão explícita do usuário para acessar recursos sensíveis como câmera ou localização.

Abaixo, alguns exemplos de uso:

## Geolocation
```JavaScript
// Verificar se o navegador suporta geolocalização
if ("geolocation" in navigator) {
  navigator.geolocation.getCurrentPosition(
    (position) => {
      document.getElementById("mapa").innerHTML = `
        <p>Você está em: ${position.coords.latitude}, ${position.coords.longitude}</p>
      `;
    },
    (error) => {
      alert("Não foi possível obter sua localização");
    },
    {
      enableHighAccuracy: true, // Maior precisão
      timeout: 10000, // 10 segundos de timeout
      maximumAge: 60000 // Cache de 1 minuto
    }
  );
} else {
  alert("Geolocalização não suportada neste navegador");
}
```

## Fetch
```JavaScript
async function getUsers() {
  try {
    const response = await fetch('https://webpage.com/users');
    
    if (!response.ok) {
      throw new Error(`Erro HTTP: ${response.status}`);
    }
    
    const users = await response.json();
    showUsers(users);
  } catch (error) {
    console.error('Falha ao buscar usuários:', error);
  }
}
```
