Leiautes modernos em CSS
=========

**Flexbox** e **Grid** são as principais técnicas de layout em CSS moderno, cada uma com seu propósito específico.

## Flexbox

Unidimensional, ideal para organizar elementos em uma única direção - seja linha ou coluna. Alguns exemplos de componentes que se servem muito bem dele são menus de navegação, barras de ferramentas e cards. Sua força está em distribuir espaço inteligentemente entre elementos e centralizar conteúdo com propriedades como `justify-content` e `align-items`.

## Grid

Bidimensional, controla simultaneamente linhas e colunas. É perfeito para layouts complexos como estruturas completas de página, galerias e dashboards. Com Grid você define áreas específicas usando `grid-template-areas` e cria grades responsivas com `repeat(auto-fit, minmax())`.

A grosso modo, Grid é usado para estruturas importantes como a base de estilo que define o layout do site e Flexbox é utilizado nos componentes internos. Eles são complementares: um site pode ter um container Grid definindo header, main e sidebar, enquanto dentro dessas áreas usamos Flexbox para alinhar menus e conteúdo.

Exemplo simples de uso, abaixo o código em HTML:

```html
<div class="container"}>
    <header class="header"}>Header</header>
    <main class="main"}>Content</main>
    <nav class="sidebar"}>
        Sidebar
    </nav>
    <footer class="footer"}>Footer</footer>
</div>
```

Em seguida a estilização feita em código CSS:

```css
.container: {
    display: 'grid';
    grid-template-areas:
        'header header header header'
        'main main . sidebar'
        'footer footer footer footer';
    grid-template-columns: '1fr 1fr 1fr 1fr';
    grid-template-rows: 'auto';
    gap: '8px';
}

.header: {
    grid-area: 'header';
    display: 'flex';
}

.main: {
    grid-area: 'main';
    display: 'flex';
}

.sidebar: {
    grid-area: 'sidebar';
    display: 'flex';
}

.footer: {
    grid-area: 'footer';
    display: 'flex';
}
```
